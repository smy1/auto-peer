# Automatic transcription of audio files
File list:
- [audio2xlsx.ipynb](./audio2xlsx.ipynb) Automatic transcription script: Opens Google Colab to transcribe .mp3 files into .srt using Whisper, then converts it to .xlsx using Python.
- [051_sbr.mp3](./051_sbr.mp3) Audio file: An example of mp3 file that can be input into the automatic transcription script.
- [051_transcript.xlsx](./051_transcript.xlsx) Original transcript: An example output of the automatic transcription script.
- [051_forcoding.csv](./051_forcoding.csv) Edited transcript: An example of a transcript that is edited to add "speaker" information and converted into csv for later processing.
